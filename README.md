# 📱 Mobile Price Prediction

## 📌 Overview

This project predicts mobile phone prices based on features like RAM, storage, and battery.

## 📊 Features

* RAM
* Storage
* Battery
* Camera
* Screen Size

## 🤖 Model

* Linear Regression

## 📈 Result

* R² Score ≈ 0.61

## 🛠️ Tools

Python, Pandas, Matplotlib, Seaborn, Scikit-learn
